# Automação de Arquivos em Python

Uma ferramenta utilitária via terminal desenvolvida em Python para organizar, mover e renomear arquivos em massa.

---

## Funcionalidades

O script oferece um menu interativo com as seguintes opções:

1. **Mudar o Diretório:** Permite alterar a pasta de trabalho atual em tempo real.
2. **Renomear Arquivos:** Renomeia todos os arquivos da pasta sequencialmente (ex: `Foto_1.jpg`, `Foto_2.jpg`).
3. **Renomear por Extensão:** Altera o nome apenas de arquivos com uma extensão específica (escolhida via menu).
4. **Mover Arquivos (Organizador):** Cria pastas automaticamente (`Imagens`, `Documentos`, `Musicas`, `Videos`) e move os arquivos para suas respectivas "gavetas" com base na extensão.
5. **Sair:** Finaliza o programa com segurança.

---

2. Baixe o arquivo do projeto e abra o terminal na pasta dele.
